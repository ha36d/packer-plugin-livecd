# livecd

Type: `livecd`

<!--

The livecd post-processor is used to export Packer livecd builds.


<!-- Post-Processor Configuration Fields -->

### Required

- `mock` (string) - The output path where to save exported build to.

### Example Usage


```hcl
 source "livecd" "example" {
   mock = "jay"
 }

 build {
   sources = ["source.livecd.example"]

   post-processor "livecd" {
     mock = "builds/livecd.box"
   }
 }
```
